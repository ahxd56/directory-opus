## DO 到底是什么来头？

Directory Opus 是澳大利亚 GP Software 公司的产品，历史比很多人用电脑的年纪还长——1990 年第一版发布，原本是为 Amiga 系统开发的。能从那个年代活到现在，还持续更新到 v13.22（2026 年 3 月刚更新），说明它不是靠营销撑着的，是靠真材实料。

现在它在 Windows 平台上的定位就是：**资源管理器的专业替代品**。装好之后，你按 Win+E 打开的就是 DO，双击文件夹也是 DO，原来的右键菜单（包括 Git Bash Here 之类的）也都能自动迁移过来。对使用习惯的冲击几乎为零，但操作效率提升是实打实的。

## 和 Total Commander 比怎么样？

经常有人拿这两个放在一起问。其实使用感受差距挺大：

Total Commander 功能不输，但上手成本高——那套快捷键体系对很多人来说需要专门花时间去学，操作逻辑和普通鼠标用户的直觉差距比较大。

DO 则是另一个风格：界面和资源管理器长得像，多标签、双窗格、自定义工具栏，上手不需要特别学习，鼠标用户直接就能驾驭，但功能深度丝毫不差。所以常见说法是"TC 是效率狂人的工具，DO 是更多数人的工具"——不算贬低 TC，只是受众定位不同。

## 核心功能一览

**双栏视图 + 多标签**：两个窗格同时打开不同目录，左边源文件夹右边目标文件夹，拖拽复制不用来回切换。多标签像浏览器一样管理多个路径，文件整理效率翻倍。

**批量重命名**：支持正则、编号、替换字符串等多种规则，设计师和摄影师处理大量素材基本靠这个功能。

**文件颜色标记与标签**：给不同类型或状态的文件标上颜色，一眼认出，不用逐个打开确认。

**高级过滤与排序**：按文件属性、修改日期、大小范围等条件筛选，目录里几百个文件找特定类型秒出结果。

**内置预览**：不用打开文件就能预览内容，图片、文档、代码都支持。

**重复文件查找 + 文件夹同步**：整理硬盘、备份数据都用得上。

**脚本插件 IDE**：有折腾欲的用户可以写脚本定制几乎任何操作，社区里已经有大量现成插件可以直接用。
